<h1>Lights Out!</h1>

<h1>Description:</h1>
Puzzle game where the objective is to make all the numbers match.
Clicking one button will make all adjacent buttons switch on or off.
Try to match all the numbers in the least number of moves!

<h1>Features:</h1>
Four different game modes
